procedure setCellFont ( Integer , Integer , String , Integer , Integer , Boolean , Boolean )
Muda a fonte de uma célula
Se "aFontName" = '' valerá a fonte corrente
Se "aSize" = 0 valerá o tamanho corrente
Parâmetros:
- Linha e Coluna
- Nome da Fonte
- Tamanho
- Cor
- Negrito (boolean)
- Itálico (boolean)
