**************** Classes de Terceiros - Auxiliares independentes ***************

Simulation
  - TSimulation
    Classe que encapsula o mecanismo genérico de simulação computacional baseado em
    eventos e passagem de tempo.

    Metodos
      procedure Execute
      procedure Stop

    Propriedades/Eventos
      OnBeforeExecute : procedure
      OnAfterExecute  : procedure
      OnClock         : procedure
      OnEvent         : procedure

drPascal
  - TPascalScript
    Classe que reconhece/compila/executa scripts baseados na linguagem Object Pascal

    Metodos
      procedure Compile
      procedure Execute

    Propriedades
      Text      : TStrings
      Lib       : TLib
      Variables : TVariables
      Compiled  : Boolean

Lib_GlobalObjects
  - TGlobalObjects
    Classe que encapsula o mecanismo de gerenciamento de Variáveis Globais (instâncias
    de objetos) acessíveis via programação direta ou indireta por Script

    Metodos
      procedure Add(Name: String; Object: TObjects)
      procedure Remove(Name: String)
      function  ObjectByName(Name: String): TObject

    Propriedades
      Count: Integer

wsMatrix
  - TwsMatrix
    Classe que encapsula a funcionalidade de matrizes matemáticas e
    conjuntos de dados (objetos parecidos com tabelas de banco de dados)

    Metodos
      constructor Create(nRows, nCols: Integer)
      procedure Add(Row: TwsVec)
      procedure Remove(Row: Integer)

    Propriedades
      Data[i, j]: Real
      nRows     : Integer
      nCols     : Integer

wsVec
  - TwsVec
    Classe que encapsula a funcionalidade dos vetores

    Metodos
      constructor Create(Len: Integer)
      procedure Add(x: Real)
      procedure Remove(i: Integer)

    Propriedades
      Data[i]: Real
      Len    : Integer
