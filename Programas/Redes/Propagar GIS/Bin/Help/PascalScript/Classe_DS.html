Classe TDelphiScript 

  TDelphiScript = Class
  public
    constructor Create;
    destructor Destroy; override;

    function  getStrVarType(T: TVariableType): String;

    function  Compile: Boolean;
    procedure Execute;
    procedure Stop;

    property Text : TStrings;
    
    property Scanner        : TPascalScanner;
    property ByteCode       : TByteCode;
    property VirtualMachine : TVirtualMachine;

    property Objects   : TdrObjectList;
    property Functions : TFunctionList;
    property Procs     : TProcList;

    property Errors   : TStrings;
    property Compiled : Boolean;
    property GerCODE  : Boolean;
  end;
Métodos

function Compile: Boolean;

Caso nenhum erro seja encontrado no texto do script, um p-code (executável) será gerado. Este p-code será armazenado na propriedade ByteCode.

procedure Execute;

Se o script ja foi compilado, o p-code será executado, caso contrário ele será compilado e depois executado.

procedure Stop;

Termina a execução de um script.

function getStrVarType(T: TVariableType): String;

Retorna em forma de String o tipo de uma variável.

Propriedades

Text : TStrings;

Contém o script a ser executado

Scanner : TPascalScanner;

Responsável por separar o script em pedaços (Tokens). Controla o acesso aos tokens pelo compilador.

ByteCode : TByteCode;

Responsável por armazenar e gerenciar o p-code. É usado pela máquina virtual para executar o script.

VirtualMachine : TVirtualMachine;

Máquina virtual responsável por executar o p-code.

Objects : TdrObjectList;

Mantém uma lista com as características dos objetos que podem ser criados através do Script.

Functions : TFunctionList;

Mantém uma lista com as características das funções que podem ser chamadas pelo Script.

Procs : TProcList;

Mantém uma lista com as características dos procedimentos que podem ser chamados pelo Script.

Errors : TStrings;

Mantém uma lista com as mensagens de erro da última tentativa de compilação

Compiled : Boolean;

Informa se o Script foi compilado com sucesso ou não

GerCODE : Boolean;

Informa ao compilador se deve ou não gerar o p-code.
