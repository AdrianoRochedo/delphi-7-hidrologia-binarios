Abrindo, Editando e Fechando um Projeto Existente

A abertura de um Projeto existente poderá ser feita do seguinte modo:

  • Escolher a opção Abrir (Ctrl + A) do Menu Arquivo ou o ícone correspondente na Barra de Ferramentas Principal.

    • Esta opção abrirá imediatamente uma janela de seleção de arquivos. O IPHS1 memorizará sempre o último Diretório de Pesquisa utilizado. A partir dele ou de outro que poderá ser selecionado, o usuário poderá abrir o arquivo desejado.

    Este procedimento abrirá imediatamente a janela Área de Projeto do arquivo selecionado que, então, poderá ser objeto de edição ou de qualquer procedimento permitido pelo IPHS1.

    O usuário poderá ter vários Projetos abertos simultaneamente.

    Quando desejar encerrar um Projeto o usuário deverá:

  • Escolher a opção Fechar do Menu Arquivo.

    • Esta opção irá fechar o Projeto ativo. Caso o Projeto não tenha ainda sido salvo ou caso tenha sofrido alguma alteração o IPHS1 perguntará ao usuário de deseja fazê-lo.
